# coc-vim-options

Set vim options from coc-settings.json

## Install

`:CocInstall coc-vim-options`

## Keymaps

`nmap <silent> <C-l> <Plug>(coc-coc-vim-options-keymap)`

## Lists

`:CocList demo_list`
